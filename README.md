# Aura Ecommerce

## Overview

Aura Ecommerce is a full-stack e-commerce web application developed as part of the Aurashine Recruitment Assessment.

The application allows users to browse products, view product details, manage a shopping cart, and place orders through a responsive user interface built with Angular and a backend powered by Node.js, Express.js, and MongoDB Atlas.
